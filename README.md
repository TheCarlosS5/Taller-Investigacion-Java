# Taller de Investigación - Métodos Abreviados y Sintaxis Comprimidas en Java

Este repositorio contiene los entregables correspondientes al taller de investigación sobre la evolución de la sintaxis y características modernas de Java.

## Contenido del Repositorio

*   **[Taller_Investigacion_Java.pdf](Taller_Investigacion_Java.pdf)**: Informe final en formato PDF redactado en tono de aprendizaje. Incluye la investigación y análisis de 20 temas de sintaxis en Java (15 del listado propuesto y 5 autónomos).
*   **[Taller_Sintaxis_Java_v2.docx](Taller_Sintaxis_Java_v2.docx)**: Archivo fuente de Word utilizado para la exportación.
*   **[crear_documento_investigacion.py](crear_documento_investigacion.py)**: Script en Python con el cual se automatizó el formateo y generación del documento Word.
*   **[ejemplos-java/](ejemplos-java/)**: Carpeta que contiene los ejemplos de código funcionales en Java para 5 temas clave:
    1.  *Operador Ternario* (`OperadorTernarioDemo.java`)
    2.  *Inferencia con var* (`InferenciaVarDemo.java`)
    3.  *Expresiones Lambda* (`ExpresionesLambdaDemo.java`)
    4.  *Clase Optional* (`ClaseOptionalDemo.java`)
    5.  *Switch Expressions* (`SwitchExpressionDemo.java`)
